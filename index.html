Facial Recognition on Node express server.

In this experiment, I have utilized the Javascript version of Opencv for Node Express server to do facial recognition of images obtained from the user's webcam through an HTML5 browser. This is then used for authenticating a user for login or sign-up into a particular website.

Authors and Contributors

The general authentication flow was obtained from https://github.com/mountain-device/easyauth and the Facial recognition component is being developed by Supriya @suprbh.
